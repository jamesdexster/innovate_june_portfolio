EM

EM - The font is relative to its parent font size.

On this page, the body font is set to 2.5em - 2.5x the size of the browser's default size. H1's parent is body - so heading 1 is 1.5 x the body size!
Content is set to 1em - and its parent is body, so content is the size of body.
